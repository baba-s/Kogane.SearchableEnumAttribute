# Kogane Searchable Enum Attribute

Inspector で列挙型が検索できるようになるエディタ拡張

## 使用例

```cs
using UniSearchableEnumAttribute;
using UnityEngine;

public class Test : MonoBehaviour
{
    [SearchableEnum] public KeyCode m_keyCode;
}
```
